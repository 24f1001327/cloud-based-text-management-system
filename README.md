# Cloud-Based Text Management System

## Overview
This project is a cloud-hosted web application designed to store, edit, and manage text content. The application was developed incrementally, with features added as new requirements and simulations were introduced.

## Key Features
- User authentication (login system)
- Rich text editing using TinyMCE
- Image upload and content editing
- Secure data storage using PostgreSQL
- Backend built with Flask
- Frontend using HTML, CSS, and JavaScript

## Tech Stack
- Backend: Flask (Python)
- Frontend: HTML, CSS, JavaScript
- Editor: TinyMCE
- Database: PostgreSQL
- Cloud Platform: AWS

## Project Evolution
The system was upgraded over time to support additional editing features and improved data handling as new simulations were added.

## Skills Demonstrated
- Full-stack web development
- Database design and integration
- Cloud deployment fundamentals
- Secure authentication workflows
- Structured and iterative problem-solving

## Notes
Sensitive information such as credentials, keys, and real user data has been removed to maintain security.
